# ATLAS AGV — Professional Reports

## Files

| File | Description |
|------|-------------|
| `ATLAS_AGV_SIMULATION_TO_PHYSICAL_CONVERSION_REPORT.md` | Pandoc Markdown — Sim-to-Physical conversion guide |
| `ATLAS_WAREHOUSE_AGV_COMPLETE_ENGINEERING_DISSERTATION.md` | Pandoc Markdown — Full B.Tech dissertation |
| `convert_reports.sh` | Bash script to generate DOCX + PDF from Markdown |

## Quick Start

### Generate DOCX and PDF

```bash
# Install prerequisites
sudo apt install pandoc texlive-latex-recommended texlive-latex-extra texlive-xetex

# Run conversion
cd ~/path/to/reports/
chmod +x convert_reports.sh
./convert_reports.sh
```

### Alternative: Manual Pandoc Commands

```bash
# DOCX only (no LaTeX needed)
pandoc ATLAS_AGV_SIMULATION_TO_PHYSICAL_CONVERSION_REPORT.md -o Report1.docx --toc --number-sections

pandoc ATLAS_WAREHOUSE_AGV_COMPLETE_ENGINEERING_DISSERTATION.md -o Report2.docx --toc --number-sections

# PDF (requires texlive)
pandoc ATLAS_AGV_SIMULATION_TO_PHYSICAL_CONVERSION_REPORT.md -o Report1.pdf --pdf-engine=xelatex --toc --number-sections

pandoc ATLAS_WAREHOUSE_AGV_COMPLETE_ENGINEERING_DISSERTATION.md -o Report2.pdf --pdf-engine=xelatex --toc --number-sections
```

### Alternative: Use Google Docs

1. Generate DOCX using pandoc (no texlive needed)
2. Upload DOCX to Google Docs
3. Format as needed
4. Download as PDF
